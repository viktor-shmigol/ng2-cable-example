Ng2-cable-example
-

API ng2-cable: [https://github.com/viktor-shmigol/ng2-cable](https://github.com/viktor-shmigol/ng2-cable)

Setup
-

    git clone
    bundle install
    rails db:create db:migrate
    rails s

    npm install
    npm start
